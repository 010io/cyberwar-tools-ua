# Squad303 Automation Guide

## Overview
Squad303 is a Python automation framework for TextNow SMS operations supporting counter-propaganda efforts.

## Installation
```bash
pip install -r squad303-automation/requirements.txt
```

## Configuration
Edit `.env` file with TextNow credentials and webhook settings.

## API
POST /webhook - Send automated messages
GET /health - Check bot status
GET /stats - Get operational statistics

## Security
- Use HTTPS/TLS in production
- Implement rate limiting
- Monitor API usage
