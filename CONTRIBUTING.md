# Contributing to CyberWarTools UA

Welcome! We appreciate your interest in contributing to this Ukrainian cyber operations toolkit.

## How to Contribute

### 1. Fork & Clone
```bash
git clone https://github.com/YOUR_USERNAME/cyberwar-tools-ua
cd cyberwar-tools-ua
git checkout -b feature/your-feature
```

### 2. Make Changes
- Follow existing code style
- Add comments for complex logic
- Test your changes

## Code Standards

### Python
- Use `black` for formatting: `black .`
- Use `bandit` for security: `bandit -r .`
- Add docstrings to functions
- Type hints appreciated

### JavaScript
- Use ESLint
- Consistent naming conventions
- Comments for complex logic

## Security

- Never commit `.env` files
- Don't include API keys
- Report vulnerabilities to security@010io.dev
- See [SECURITY.md](SECURITY.md) for details

## Legal & Ethics

By contributing, you agree that:
- Your contributions will be licensed under MIT
- You respect all legal boundaries
- Content is for Ukrainian cyber defense only
- No harassment or spam
